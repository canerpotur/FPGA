The aim of this project is displaying an image on a VGA monitor using FPGA. 
I used 240x320 images in hex format.

Modules: top module-->vgaaa: Introducing inputs, outputs, submodules. 
         submodule--->vga_controller: Includes the codes for reading an image and displaying it on VGA screen.


How to run: -->Copy the verilog codes in this folder to "INTRODUCTION TO VGA\FULL PROJECT\vgaaaa" and "INTRODUCTION TO VGA\FULL PROJECT\vga_controller".
            -->Copy the images(in hex format) to "FULL PROJECT\vga_controller". 
            -->Run.
